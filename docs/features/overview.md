# Features

In Ductape, **features** are workflow orchestrations: multi-step logic that composes actions, databases, notifications, brokers, and other product resources.

Features are defined in code (TypeScript SDK reference) and configured per product and environment. Java, Go, and .NET SDKs support workflow **execute**, **dispatch**, and local executor seams — see [SDK readiness](/sdk-readiness).
